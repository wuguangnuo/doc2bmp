# doc2bmp

------

需求：将云桌面中的文件拷贝出来。（云桌面可以读取本地磁盘，反之不可以）
思路：云桌面下，将文件转换成图片；本机截图，并将图片还原成文件
功能：实现所有格式的文件与BMP格式图片间的转换

使用方法：
